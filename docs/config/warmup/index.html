Warmup Script

Note

This script is optional. In most cases, environments will be spawned successfully without it.

The warmup script is executed from the context of the dev container before building and deploying the rest of the environment.
This script can help you prepare whatever needs to be prepared for the containers build to be successful.

Remember

Add all warmup script dependencies to the dev container using its Dockerfile.
