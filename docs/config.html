Configuration

📄️ raftt.yml

The configuration of Raftt is located in the raftt.yml file that is created the first time you run raftt up for the repository.

📄️ Dev Container

In addition to the containers described in your docker-compose file, another container is created when an environment is spawned - the dev container (named dev by default).

📄️ Warmup Script

The warmup script is optional.

📄️ Database Seeding

When spawning environments, you might want to populate your databases with data before using the environment.
